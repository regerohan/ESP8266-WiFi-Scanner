# ESP8266-WiFi-Scanner
A WiFi Scanner for ESP12-E

### What is this repository for? ###

This script scans what networks are accessible by the ESP so that you don't waste time.

### How do I get set up? ###

* Clone the repo
* Import Project in PlatformIO 
* Enjoy
